# FINVA Onboarding — Data Entry Guide

Welcome! Please fill in these 4 spreadsheets with your client data so we can
load it into FINVA for your Beta trial. It usually takes 1–3 hours depending on
how many clients you have.

## The 4 files

| File | What goes in it |
|------|-----------------|
| `1_Clients_Database.xlsx` | One row per **client** (name, status, risk profile, AUM, income) |
| `2_CashFlow_Database.xlsx` | One row per client per month (income vs expenses) — optional |
| `3_Portfolio_Holdings.xlsx` | One row per **investment holding** (unit trust, EPF, etc.) |
| `4_Assets_Liabilities.xlsx` | One row per **asset or liability** (property, cash, car, loans) — for net-worth & planning |
| `Insurance_Policies_Template.xlsx` | One row per **insurance policy** |

## How to fill them

1. Open each file. You'll see:
   - **Row 1–2:** title / instructions (don't touch)
   - **Row 3:** the **column headers**
   - **Row 4:** a hint for each column (what to type)
   - **Row 5 onward:** ← **start typing your data here**, one record per row
2. Only type into the columns that have headers. Leave blanks if you don't have a value — don't put "N/A".
3. Save the file when done (keep it as `.xlsx`).

## ⚠️ The 3 rules that matter most

1. **Names must be IDENTICAL across all files.**
   If a client is `Tan Wei Ming` in the Clients file, they must be spelled
   exactly `Tan Wei Ming` in the Portfolio and Insurance files too (same
   spelling, same spacing). This is how FINVA links everything together.

2. **Insurance: "Policy Owner" vs "Life Assured" are different people.**
   - **Policy Owner** = the client who owns/pays the policy (this is who it's grouped under).
   - **Life Assured** = the person actually insured (can be the owner, a spouse, or a child).
   - Example: Policy Owner `Tan Wei Ming`, Life Assured `Tan Xiao Ming` (his son).

3. **Numbers and dates:**
   - Money: digits only — type `250000`, not `RM 250,000` or `250k`.
   - Dates: use the format shown in the row-4 hint (e.g. `2026-05-01`).

## Quick tips

- **Status** columns: use the options listed in the row-4 hint (e.g. Active / Lapsed / Matured).
- **Medical Card** (insurance): put room & board and annual limit together, e.g. `200/day · 1.5M`.
- **Assets & Liabilities**: do NOT re-list investment holdings here (they're already in the Portfolio file and would be double-counted). List property, cash/FD, EPF, vehicles, business value, and all loans/debts. "Type" must be exactly `Asset` or `Liability`.
- You don't need to fill the CashFlow file to get started — Clients + Portfolio + Insurance is enough.
- Don't add, rename, or reorder columns — just fill rows.

## When you're done

Send all 4 files back to your admin. They'll import them and create your FINVA
login. Your data will be private to you (only you and the admin can see it).

Questions? Ask your admin before guessing — it's easier to fix before import.
